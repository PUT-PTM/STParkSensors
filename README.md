# STParkSensors

Overview - University Project. Main idea is to program uC as a ParkSensor. 

Description - Used hardware: STM32F407VGT, HC-SR04 2-200cm.

Tools - Used software is System Workbench for STM32, C language project.

How to run - For now, just compile and run.

How to compile - Default C language compilator for STM32 architecture.
